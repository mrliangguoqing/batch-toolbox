# Batch-Toolbox (Windows 批处理脚本集)

收集实用的 Windows `.bat` 脚本，旨在通过简单的双击操作，替代复杂的命令行输入或重复的手动操作。

## 脚本介绍

### keil_cleaner.bat

功能：用于 MDK-ARM(µVision5) 的文件清理，一键移除项目中的中间文件、日志文件、调试文件等，极大缩小项目体积，方便打包分享。

使用说明：将 keil_cleaner.bat 文件复制到 MDK-ARM 工程路径下（参考[此工程](https://github.com/mrliangguoqing/stm32h7-elderly-health-monitor)），双击运行即可。
